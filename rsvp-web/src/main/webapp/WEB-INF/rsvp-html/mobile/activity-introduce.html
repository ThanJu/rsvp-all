全球新能源汽车&电池创新与技术峰会2019将汇集来自新能源汽车整车商、电池厂商、电机电控生产商、BMS系统服务商、Pack&电芯厂商、汽车零部件商、充电桩、充电站解决方案供应商、系统集成商、设备供应商、电池及汽车材料企业、新能源汽车平台，检测咨询机构，科研机构及院校等等​超过600位来自全球的行业嘉宾齐聚一堂，以“求变，突围，成功”为核心主题，100+新能源汽车及电池产业顶尖专家演讲分享包括最新政策与全球发展趋势，商业模式—可持续发展，老牌车企与造车新势力的竞争与合作，工业4.0时代，最新技术包括电机电控创新技术、充电桩、新能源汽车平台、电池前沿技术与趋势、动力电池安全、用材轻量化、燃料电池、电池材料&新型材料、BMS、电池智能制造以及储能电池等热点话题。本次峰会也是一个理想的产业交流和合作的平台，发现新的客户，找到理想中的合作伙伴，与行业内最优秀的高管与嘉宾进行现场沟通
